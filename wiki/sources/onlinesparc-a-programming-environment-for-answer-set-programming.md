Paper presenting onlineSPARC, a browser-based development environment for ASP/SPARC designed to lower setup barriers and improve educational usability through querying and visual output support.

## Citation

Marcopoulos, E., & Zhang, Y. (2018). onlineSPARC: a Programming Environment for Answer Set Programming. arXiv:1809.08304v1.

## Problem and aim

Teaching and learning ASP can be hindered by tool installation complexity and by less intuitive presentation of answer sets, especially for newcomers and school-level learners. The paper introduces an online IDE to streamline coding, solving, querying, and visual rendering.

## Core contributions

- Provides a web-based ASP/SPARC environment with integrated editing, file management, query handling, and answer-set generation.
- Uses SPARC (with explicit sorts) to support clearer educational modeling and earlier error detection.
- Adds drawing/animation predicates to make ASP results more intuitive and engaging.
- Reports practical classroom/outreach usage where tooling overhead was reduced.

## Relevance to this wiki

- Extends ASP coverage from theory/applications into practical tooling and adoption pathways.
- Useful for student-project and pedagogy-oriented planning where ASP is part of implementation tracks.
